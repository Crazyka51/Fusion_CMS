# Salon Zuza - Obrázky

Tato složka obsahuje obrázky pro Salon Zuza:

## Struktura obrázků:
- `salon-interior-*.jpg` - Interiér salonu
- `damsky-strih-*.jpg` - Ukázky dámských střihů  
- `panske-strih-*.jpg` - Ukázky pánských střihů
- `barveni-*.jpg` - Ukázky barvení vlasů
- `kosmetika-*.jpg` - Kosmetické ošetření
- `team-*.jpg` - Fotografie týmu

## Použití:
Obrázky se načítají z `/imgssalon/` cesty v aplikaci.
Doporučené rozměry: 1200x800px pro galerii, 600x400px pro náhledy.

## Formáty:
Podporované formáty: JPG, PNG, WebP
Optimalizované pro web s kompresí kvality 85%.
